## 패턴 분석

### Keep (유지)
- 사용자 스크린샷 기반 버그 사냥 → 즉시 코드 추적 → 수정: 효율적
- Explore 에이전트로 다중 파일 동시 조사: 빠른 원인 파악
- Codex 교차 검증 → HIGH 보안 이슈 발견: 자기 편향 없이 구조 문제 식별

### Drop (중단)
- 없음 — 세션이 짧고 집중됨

### Try (시도)
- 프롬프트 캐싱 효과를 실제 API 비용으로 검증 (A/B 비교)
- 한국어 조사 스트리핑 정확도를 형태소 분석기로 향후 개선 가능
